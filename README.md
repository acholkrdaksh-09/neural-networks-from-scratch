# neural-networks-from-scratch

Overview:-
This repository documents my journey of learning and implementing Neural Networks from scratch. 
Instead of relying on deep learning frameworks like TensorFlow or PyTorch from the beginning, the goal
is to understand every mathematical concept, algorithm, and software component that makes modern AI possible.

Every implementation is built step by step using Python, with an emphasis on understanding the underlying theory before using high-level libraries.
